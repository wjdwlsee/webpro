 변수란 프로그램 작업을 처리하기 위해 하나의 값을 저장할 수 있는 메모리 공간을 말하며 임의의 메모리 공간에 이름을 붙여 관리하는 것이다.
 자바에서는한가지 타입만 값으로 저장될 수 있고 식별자(identifier)라고도 하며
자바코드에서 변수로 입력시킨 이름을 의미한다. 
 변수 선언 방법은 다음과 같다.
int       i         =         10;      //선언과 할당
 데이터 타입의 키워드는 boolean, char ,byte,short,int,long,float,double이 있고 자료형(논리형,문자형,정수형,실수형) , 크기, 표현범위가 다 다르다.
 기본 데이터 type (primitive 기본 자료형)이란 메모리에 있는 실제값 = 변수 데이터 값 즉, Java 언어에 이미 존재하고 있는 데이터 타입. 주로 간단한 데이터들 말한다.
 참조 데이터 type (Object, 객체 자료형) 란 메모리에 있는 실제값(stack영역) = 변수가 저장된 주소(실제 변수 데이터는 heap 영역)이고 여러 가지 데이터들이 모여 있는 복잡한 데이터로 기본 자료형에 비해 크기가 크다
 형변환에는 묵시적 형변환(작은 주머니의 데이터를 큰 주머니로 옮기는 것)과 명시적 형변환 (명시적 형변환 : 큰 주머니의 데이터를 작은 주머니로 옮기는 것. 데이터 손실이 있을 수 있으며 코드에 명시해 주어야 한다)이 있다.
 연산자란 일정한 규칙을 가지고 계산하는 것으로 종류에는 산술연산(*,/,+,-, %), 증감연산(++, --), 동등관계(비교)연산(== ,!=, <,<=,>,>=), 삼항연산(?,:), 할당연산자(=,+=,-=,*=,/=,%=)이 있다.
연산자에는 우선 순위가 있으며 결합성도 각자 다르다.
