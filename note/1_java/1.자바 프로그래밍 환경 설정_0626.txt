컴퓨터 프로그래밍이란 언어를 이용해서 프로그램을 만드는 행위이다.
java의 개발 환경 설정은 JDK, IDE를 다운로드하여 실행한다.
java의 프로그램 실행단계는 java 소스 파일을 작성 후 컴파일러로 바이너리 코드를 생성, jvm 구동명령어로 실행한다.
컴파일러는 java 파일을 jvm이 받아들일 수 있는 class 파일로 변환시켜주는 프로그램이다 
파일을 어디서나 실행하기 위해 환경변수 설정을 해주어야하는데 위의 과정이 필요하다 
시스템변수를 새로만들고 path에서도 내용을 추가 해준다. 
java 사용하여 이클립스를 통해 쉽게 "Hello,java" 를 실행하였다.
