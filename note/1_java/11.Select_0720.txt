SQL 문장 작성은 대소문자를 구별하지 않지만 키워드는 대문자로 입력하고 키워드를 제외한 다른 모든 단어, 즉 테이블 이름, 열 이름은 소문자로 입력하기도 한다.
문장은 한 줄 또는 여러 줄에 입력될 수 있으며 줄을 나누고 탭과 줄 넣기를 코드를 보다 쉽게 읽기 위해 권장한다.
키워드를 제외한 다른 모든 단어, 즉 테이블 이름, 열 이름은 소문자로 입력하기도 한다.
명령어의 끝은 ; 을 사용하고 주석처리는 --을 사용한다.
특정 열만 출력할 때에는 WHERE절 (조건절)을 쓰며 비교연산자에는 같다(=),다르다(!=,^=,<>),>,>=,<,<=이 있으며 숫자형(NUMBER), 문자(VARCHAR2), DATE모두 사용 가능하다. 또,조건절의 논리연산자는 AND, OR, NOT 이 있다.
날짜 표기법은 (YY/MM/DD) 또는 (RR/MM/DD)이 있는데 어떤 세팅에서든 사용할 수 있도록 TO_CHAR 을 쓰기도 한다.
필드나 문자들을 연결하기 위해 사용하는 연결연산자(||)가 있다
결과를  정렬(오름차순, 내림차순)하기 위해 ORDER BY절을 쓰며 내림차순은 DESC을 붙인다.
SQL연산자에는 또 BETWEEN, IN(A,B), LIKE,IS NULL이 있다.
BETWWN A AND B 은A부터 B까지의 값을 나타내며 A가 꼭 B보다 커야하며 , list의 값 중 어느 하나와 일치하면 IN(LIST)를 사용한다.
LIKE pattern 은 검색하고자 하는 값을 정확히 모를 경우 pattern과 일치하는지를 검색하고, 두가지의 와일드 카드가 사용되는데 % 은 %(0글자이상),_(한글자)를 포함한다.
IS NULL은 NULL 값을 가졌다 , NOT +연산자는 연산자가 일치하지 않는다를 말한다.
