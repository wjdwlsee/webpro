이벤트란 키보드를 이용해 버튼을 입력하거나 마우스 클릭과 같이 다른 것에 영향을 미치는 것. 자바 스크립트에는 다음과 같은 이벤트가 있다
마우스 이벤트, 키보드 이벤트, HTML 프레임 이벤트, HTML 입력 양식 이벤트, 유저 인터페이스 이벤트, 구조 변화 이벤트, 터치 이벤트 
(ex) onclick, onmouseover, onmouseout, onkeyup(키보드의 특정키를 눌렀다가 올라올 때, event.keyCode 해당키의 고유 코드값..)
이벤트를 연결한다. Window 객체의 onload 속성에 함수 자료형을 할당하는 것, load는 이벤트 이름, 이벤트 타입 ,header 객체의 이벤트 이름은 click, 이벤트 속성은 onclick, 이벤트리스너는 whenClick()함수
문서 객체의 이벤트를 연결하는 방법을 이벤트 모델이라고 하는데 종류에는 인라인 이벤트 모델, 기본 이벤트모델, 표준 이벤트 모델이 있다.
이벤트 전달 ; 어떠한 이벤트가 먼저 발생해 어떤 순서로 발생하는지를 이벤트 전달이라 한다. 자바스크립트는 자식 노드에서 부모 노드 순으로 이벤트를 실행한다. 이를 이벤트 버블링 방식이라 한다
event.keyCode(입력된 키 아스키코드) , event.offsetX와 event.offsetY(클릭한 곳의 offset좌표), event.screenX와 event.screenY(클릭한 곳의 screen좌표) 등의 이벤트들도 있다.
예외처리란 프로그램이 실행되는 동안 문제가 발생하면 프로그램이 자동으로 중단되는데 이러할 때 프로그램이 중단하지 않고 대처할 수 있게 처리하는 것을 예외처리라 한다.
예외가 발생하는 이유 ①잘못된 코드 작성 ②사용자가 개발자가 원하지 않는 방향으로 프로그램을 사용했을 때
오류와 예외의 차이 : try catch구문으로 해결할 수 있는 것은 예외이고 해결할 수 없는 것은 오류.
