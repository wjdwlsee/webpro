자바 스크립트의 HTML페이지의 각 태그는 웹 브라우저에 의해 순차적으로 실행되며 웹 브라우저는 head태그를 먼저 읽은 후 body 태그를 읽으므로 script 태그를 head에 넣으면 body읽기전에 실행. 반대로 body에 넣으면 head실행 후 실행한다.
자바 스크립트의 기본 문법 : 1.자료형[string, number, Boolean, object,function, undefined] ,내장함수[alert, prompt, confirm] 
2.식별자 ; 이름을 붙일 때 사용하는 단어 ex.변수명, 함수명, 속성, 메소드 : 키워드를 사용하거나 숫자로 시작하면 안됨. 특수문자는_ , $만 허용, 공백문자를 포함할 수 없음, 알파벳을 사용하는 것이 관례,각 단어의 첫글자를 대문자로,생성자함수의 이름은 항상 대문자로 시작. 그외 변수는 항상 소문자로 시작
3.주석 : /* 여려줄 사용 */  or  // 한줄만 사용. 4.출력: alert('' Hello, JavaScript '')	
5.변수의 생성과 사용: ⓐvar : 변수 선언 시 사용. 변수의 재선언 가능. 전역변수로 주로 사용 , ⓑ:let : 변수 선언 시 사용. 변수의 재선언 불가. 블록 레벨 스코프 적용, ⓒ:const : 새로운 값을 재할당이 안 된다. 객체의 경우 내부값은 바꿀 수 있다.(const arr = [1,2,3];arr[0] = -1; //객체 내부 값은 변경 가능)
6. 자료형 : 문자열 자료형(큰따옴표나 작은 따옴표를 이용가능,예외로 두가지 모두 사용도 가능) / 숫자 자료형 (정수와 유리수의 구분 없이 사용가능 하나 우선순위를 고려해야 하며 %(나머지 연산자)와 사칙연산 가능 / 불 자료형(비교연산자- !=;다르다) ,(논리연산자- !부정연산자, &&논리곱연산자, ||논리합연산자)
7 자료형 : Undefined 자료형 (변수를 선언하지 않았거나 선언했어도 초기화하지 않았을 때 undefined 자료형을 가진다)
8. 복합 대입 연산자(+= -= *= /= %=), 증감연산자, 
9 출입력: (alert 함수 : 경고 메시지를 전달하기 위한 대화상자를 표시, alert:단순히 전달사항을 출력할 때 사용하는 함수);
	 confirm 함수 : 사용자의 확인을 받기 위한 대화상자를 표시 , 반환값 = confirm(출력할 메시지); //반환값이 true면 확인단추가 false면 취소단추가 선택된 것임.
10. prompt 함수 : 사용자로 내용을 입력받을 수 있는 대화상자를 표시, 원하는 데이터를 키보드에서 입력받고 싶을 때 prompt함수를 사용,반환값 = prompt(메시지, 초기값);
