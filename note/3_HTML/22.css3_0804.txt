1. CSS3 단위들: em, %, px, pt
2. url()함수를 이용한 background-image, 로컬 폰트 지정 
3. display 속성 : 화면에 어떻게 보이는지를 설정하는 속성. 다양한 속성값이 있지만,  inline( span태그 - 높이의 속성이 없다), block(div태그 p태그 li태그 - 높이의 속성이 있다), inline-block( block의 속성을 갖고 옆으로 붙여 나가는 것 - 높이의 속성이 있다), noned 가 자주 사용된다.
4. visibility 속성 : display 속성의 none속성값과 비교하여 이해합니다. (1)display속성의 none 속성값과 비교하여 이해한다. (2)display:none은 공간도 사라짐
5. opacity 속성 : 투명도를 조절하는 속성. 여러 곳에 유용하게 사용된다
6. margin 및 padding 속성 : margin 및 padding 속성은 아주 중요한 속성으로 정확한 속성의 의미를 알아야 한다.
7. box-sizing 속성 : box-sizing속성을 이용하면 전체 사이즈의 크기를 변경 할 수 있다.
8. border 속성 : border 속성을 변경할 수 있다.
9. 글자 관련된 속성들 : font-family(글꼴), font-size(크기), font-style(이텔릭체), font-weight(진하게), line-height(행간격 조정용도나 세로축에 대한 정렬 용도), text-align(글자에 대한 정렬), text-decoration(간단한 글꼴 모양의 속성)
10. 폰트(웹폰트,로컬폰트, 시스템폰트): @font-face (1) 웹 폰트 : 인터넷만 연결되어 있다면, 다양한 폰트를 웹사이트 방문자에게 제공할 수 있도록 고안된 폰트 웹사이트를 방문하는 순간, 웹 폰트를 다운받아 웹사이트를 구성   (ex) http://fonts.google.com
				(2) 로컬폰트 (무료다운: 구글이나 네이버 폰트, 그외 사이트)
