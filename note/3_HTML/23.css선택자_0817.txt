CSS  선택자는 HTML 문서 중 특정 부분을 선택하여, 해당 부분의 속성을 변경하는 목적으로 사용한다.
tag선택자는 특정 태그를 선택하여 css속성을 적용할수 있으며 * 선택자를 사용하면 문서 전체를 선택하는 의미가 있다.
id(#)와 class(.) : 태그의 id와 clas를 이용하여 CSS에 속성을 설정할 수 있는데 혼합하여 사용할 수도 있다.
속성 선택자 는 tag의 속성을 선택하여 CSS의 속성을 설정할 수 있는데 종류는 다음과 같다.
(1) 선택자[속성] : 지정한 속성을 가지고 있는 요소를 찾아 스타일을 적용
(2) 선택자[속성=값] : 속성 값이 특정값과 같은 경우를 찾아 스타일을 적용
(3) 선택자[속성^=값] : 속성 값이 지정한 문자로 시작하는 속성값에 대해서만 스타일을 적용
(4) 선택자[속성$=값] : 속성 값이 지정한 문자로 끝나는 속성에 대해서만 스타일을 적용
후손 및 자손 선택자, 동위(=형제) 선택자(+하나만, ~다선택)도 있다.
반응 선택자 : 마우스를 올려 놓았을 때(hover상태)의 반응에 따른 속성을 설정할 수 있다
상태 선택자 : 상태에 따라 CSS속성이 변화하는 설정을 할 수 있다.  ex. input태그의 경우 focus가 맞혀졌을 때 사용한다.
구조 선택자 : 구조에 따라 CSS 속성이 변화하는 설정을 할 수 있다.
링크 선택자 : 문서에 링크(href) 되어 있는 문자를 선택하여, CSS 속성을 설정할 수 있다.
부정 선택자 : 나를 제외한 모든 태그에 CSS 속성을 적용할 수 있다
